Set
        Основная функциональность
    Хранение неповторяющихся значений в виде множества объектов.

        Примеры типичного использования
    Организация множеств. Проверка принадлежности элемента определенному множетсву.

List
        Основная функциональность
    Хранение элементов в определенном порядке следования со случайным доступом

        Примеры типичного использования
    ACL, динамические массивы

Queue
        Основная функциональность
    Хранение элементов в определенном порядке следования с последовательным доступом

        Примеры типичного использования
    LIFO

Map
        Основная функциональность
    Хранение пар элементов ключ-значение

        Примеры типичного использования
    Словари, Отображения
